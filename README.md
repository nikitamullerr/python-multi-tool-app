# Project Description
This is a beginner-friendly Python console application that combines multiple useful tools into one program. It includes a calculator, grade calculator, motivational quotes generator, and budget tracker. The project demonstrates basic programming logic, functions, and modular code structure.

---

# Features of the Project
- Basic calculator (+, -, *, /)
- Grade calculator with average score system
- Motivational quote generator
- Budget tracker (income vs expenses)
- Menu-driven interface
- Continuous loop until user exits

---

# Python Concepts Used
- Functions
- Loops (while, for)
- Conditional statements
- User input handling
- Modules (custom helpers.py, random)
- Type conversion (int, float)
- Lists
